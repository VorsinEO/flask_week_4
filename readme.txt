Пререквизиты
установите пакеты питон 3 из файла requiriments
установите и настройте переменную окружения к базе данных PostgreSQL как описано в теории недели 4

Установка
1. из папки с app,py запускаем в терминале:
    flask db init
    flask db migrate
    flask db upgrade
    появятся 5 таблиц в PostgreSQL
2. запускаем скрипт fill_goals_teachers.py чтобы загрузить в преподов и цели данные подготовленные в проекте третьей недели
    в таблицах преподов, целей и связующей их таблице появятся записи
